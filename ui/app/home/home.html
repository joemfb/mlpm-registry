mlpm

a MarkLogic Package manager

mlpm is a package manager for MarkLogic Server modules: XQuery, JavaScript, SPARQL and XSLT. Designed to be familiar, it's command-line client is heavily modeled on npm, the package manager for node.js. Packages are hosted here: a custom, searchable registry, powered by MarkLogic Server.

Install mlpm with npm (which is bundled with node.js):

$ npm install -g mlpm

See getting started for more details, or search for packages!
